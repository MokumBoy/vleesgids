ChefCuts iPhone prototype

Inhoud:
- index.html: mobiel prototype voor iPhone/PWA
- manifest.json: web app manifest
- sw.js: simpele service worker

Gebruik:
1. Open index.html in Safari op iPhone.
2. Kies 'Zet op beginscherm' om als app-achtige ervaring te gebruiken.
3. Gebruik dit prototype als startpunt voor een native SwiftUI-app of Power Apps-app.
